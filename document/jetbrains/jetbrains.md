### idea setting

#### keymap

1. ctrl + Q : 快速查注释
2. ctrl + alt + l: 格式化代码
3. ctrl + alt + o : 优化imports
4. ctrl + k : commit 代码
5. ctrl + shift + k : push 本地提交代码到远程
6. ctrl + d: 赋值一行数据到下一行
7. ctrl + shift + x: translate 中英文
8. ctrl+ / 添加或者取消注释
9. ctrl+ B 查看选中函数/类的源码
10. ctrl + alt + 左、右 回退和前进
